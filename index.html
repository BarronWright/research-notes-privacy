Privacy Policy for Research Notes

Last updated: 2025-12-30

Overview

This extension helps users save highlighted text from webpages as research notes with source information and citations, and optionally export notes to third-party services when the user requests.

Data stored locally

  • Selected text you choose to save
  • Page metadata (title, URL, access date)
  • Optional notes/tags/projects you enter
  • Settings (citation style, export template)

Data sharing

The extension shares data only when you initiate an export (e.g., to Notion, Google Docs, Word, Overleaf). It does not sell or rent user data.
